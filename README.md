# minesweeper
minesweeper game
play it here : prod version - https://cdn.rawgit.com/sktguha/minesweeper/master/minesweeper.html
or with latest master - https://rawgit.com/sktguha/minesweeper/master/minesweeper.html

To flag a mine, right click on it, or on mobile long press the button for 1.5 seconds
game is over if all mines are flagged, or you click on a mine
